## ğŸ’» Program

```
class Rectangle:
    def __init__(self, length, width):
        self.__length = length 
        self.__width = width  
    def print_inside(self):
        print(self.__length)
        print(self.__width)
    def get_length(self):
        return self.__length
    def get_width(self):
        return self.__width
r = Rectangle(5, 3)
print(r.get_length())
print(r.get_width())
```
